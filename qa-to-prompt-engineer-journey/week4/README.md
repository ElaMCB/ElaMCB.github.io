# Week 4: Portfolio, Interview Stories & Offer Filter

**Goal**: Package evidence so recruiters want to shortlist you

**Time**: ~10 hours (90-minute daily blocks)

---

## Day 1: Portfolio Site

### Build
Portfolio site (Notion/GitHub Pages) with:

### Hero Section
1-sentence value prop: "I turn vague requirements into reliable LLM features—measured, cost-optimal, shipped."

### 3 Case Cards
Each card includes:
- Problem statement
- Metric (before/after)
- Prompt delta (what changed)
- Business impact (use numbers)

### Embedded Demos
- Colab notebooks
- "Open in playground" links
- Interactive examples

### Deliverable
- Live portfolio site
- 3 case studies with metrics
- Embedded demos
- Professional design

---

## Day 2: Behavioral Stories (STAR+Data)

### Format
STAR + Data for each story:
- Situation
- Task
- Action
- Result
- **Data/metrics**

### Stories to Script (6 total)
1. "Tell me about a time a prompt failed in prod"
2. "How did you convince PM to trade latency for accuracy?"
3. "Describe a time you improved a prompt's performance"
4. "Tell me about handling a domain you weren't familiar with"
5. "How do you handle conflicting requirements?"
6. "Describe your process for evaluating prompt quality"

### Requirements
- Each story <2 minutes spoken
- Include specific metrics
- Show data-driven thinking

### Deliverable
- 6 scripted stories (STAR+Data format)
- Practice recordings (optional)
- Key metrics memorized

---

## Day 3: Mock Interviews

### Option A: Peer-to-Peer (Free)
- Join "The Prompt Community" Discord
- Find practice partners
- Schedule 45-min sessions (30 min interview, 15 min feedback)

### Option B: AI-Evaluated Solo (Free)
Record yourself on Loom answering 5 staples (2 min each):
1. "How do you detect and reduce hallucination in a RAG pipeline?"
2. "A PM wants 99% accuracy on entity extraction but won't pay for GPT-4. Walk me through your prompt/model/tradeoff pitch."
3. "Design an eval for a creative-writing assistant."
4. "Your prompt worked in playground but fails in prod—debug live."
5. "What's the difference between few-shot and Chain-of-Thought, and when would you not use CoT?"

Feed transcript into Claude/GPT-4 with meta-prompt for scoring.

### Option C: Expert Mock (Low-Cost)
- Prepfully ($30-50/session)
- Fiverr ($20) - "AI/ML technical interview mock"

### Deliverable
- 5 AI-scored answers ≥8/10
- One 45-min peer session with written feedback
- Improvement plan based on feedback
